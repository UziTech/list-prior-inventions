# list-prior-inventions
List prior inventions from github

### Usage

```
node index -u <username> -p 1
```

```
Usage: index [options]

Options:
  -u, --user <type>     <username>
  -p, --page <type>     page
  -n, --namepad <type>  pad the name
  -h, --help            display help for command
```
